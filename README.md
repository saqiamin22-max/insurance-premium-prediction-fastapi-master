# Insurance Premium Prediction API

This project is an end-to-end Machine Learning web service that predicts insurance premiums based on user demographic and health data. It features a high-performance backend and is fully containerized for production deployment.

## 🚀 Features
- **Accurate Predictions:** Uses a trained Machine Learning model to calculate premium costs instantly.
- **Production-Ready API:** Built with FastAPI for fast, asynchronous request handling.
- **Containerized:** Fully packaged using Docker to ensure it runs anywhere seamlessly.

## 🛠️ Tech Stack
- **Language:** Python
- **Framework:** FastAPI
- **Machine Learning:** Scikit-Learn
- **DevOps:** Docker
